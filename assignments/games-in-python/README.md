# 📘 Assignment: Hangman Game Challenge

## 🎯 Objective

Build the classic word-guessing game using Python strings, loops, and user input. Create a Hangman game where players guess letters to reveal a hidden word before running out of attempts.

## 📝 Tasks

### 🛠️ Build the Game Core

#### Description
Implement the main Hangman game logic that handles word selection, letter guessing, and game state management.

#### Requirements
Completed program should:

- Randomly select words from a predefined list
- Accept letter guesses from the player
- Display current progress in the format: _ _ _ (with revealed letters)
- Track incorrect guesses and remaining attempts
- End the game when the word is guessed or attempts are exhausted
- Display appropriate win/lose messages

### 🛠️ Enhance User Experience

#### Description
Add features that make the game more engaging and user-friendly.

#### Requirements
Completed program should:

- Show the list of letters already guessed
- Display the number of attempts remaining clearly
- Prevent duplicate letter guesses
- Provide helpful error messages for invalid input
- Show a final summary of the game result
